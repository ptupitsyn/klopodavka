Веб-конференции, или форумы, предназначены для свободного общения всех посетителей портала и страничек наших пользователей. Каждый зарегистрированный пользователь может создать собственный форум и поместить к себе на страничку.

Читать и писать сообщения во всех конференциях могут как зарегистированные пользователи, так и все остальные люди.

Вход в систему создания и администрирования конференций находится в меню "Форумы", доступ к которому можно получить, войдя в сеанс.

Для того чтобы вставить форум 'Klopodavka' на свою страницу Вам надо в зависимости от расширения имени файла поместить в него соответствующий текст
  1. .html .htm
    Непосредственно в HTML страницу вставить форум нельзя
  2. .dhtml
    <!--#include virtual=forum.cgi?fid=50593-->
  3. .php .php3 .php4 .phtml (только для платных пользователей на тарифных планах "Профи" и "Спутник")
    virtual("/cgi-std/forum.cgi?fid=50593&$QUERY_STRING");
  4. .shtml
    (только для платных пользователей на тарифных планах "Профи" и "Спутник") <!--#include virtual="/cgi-std/forum.cgi?fid=50593&$QUERY_STRING"-->

Для того, чтобы вы могли изменить дизайн форума по своему усмотрению, мы сделали набор стилей для каждого элемента форума. Вставьте в .dhtml документ перед тегом <BODY> следующий код:

<STYLE TYPE="text/css">
<!--
.forum_hr { color: #E0E0E0; height: 1; }
.forum_title { font-weight: bold; text-decoration: underline; color: #002952; }
-->
</STYLE>

Вот список всех стилей, используемых в форумах;

  • forum_hr -- разделительная черта. Отделяет сообщения друг от друга.
  • forum_a -- кликабельные ссылки (например, "ответить" или "показать все сообщения").
  • forum_subj -- тема сообщения.
  • forum_poster -- постер сообщения.
  • forum_date -- дата сообщения.
  • forum_message -- сообщение форума.
  • forum_title -- фраза "новое сообщение"
  • forum_form_text -- текст надписей в форме нового сообщения (Имя, e-mail, тема, сообщение).
  • forum_form_name -- стиль формы ввода имени.
  • forum_form_email -- стиль формы ввода e-mail.
  • forum_form_subj -- стиль формы ввода темы.
  • forum_form_message -- стиль формы ввода сообщения.
  • forum_form_submit -- стиль кнопки "Отправить сообщение".
